# 🚀 Monitor de Preços com Telegram Bot

Sistema completo de monitoramento de preços com alertas automáticos no Telegram.

## 📦 Funcionalidades

- 🔍 Monitoramento automático de produtos
- 🤖 Bot no Telegram com alertas em tempo real
- 🌐 Painel web para cadastro de produtos
- 💾 Banco de dados SQLite
- 🛒 Suporte para múltiplas lojas (Amazon, Mercado Livre)
- 🚫 Sistema anti-spam (evita mensagens repetidas)

---

## 🧠 Como funciona

1. Usuário cadastra o produto no painel web
2. Sistema salva no banco de dados
3. Bot verifica preços automaticamente
4. Quando o preço atinge o valor definido → envia alerta no Telegram

---

## 🛠️ Tecnologias

- Python
- Flask
- SQLite
- BeautifulSoup
- Telegram API
